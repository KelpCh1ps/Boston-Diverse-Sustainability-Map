The Boston Diverse Sustainability Map:

- Maps food banks and food assistance resources across Greater Boston  
- Allows users to quickly find food resources near them  
- Displays different resource types in interactive map layers  
- Helps communities identify areas where support may be missing  
- Uses AI-generated summaries to provide real-time updates and reviews about locations

## ğŸ› ï¸ How We Built It

We built this project using:

- Public Data from Boston Public Data  
- Google Places API to locate food distributors  
- Cleaned and standardized exported results into structured JSON format  
- Displayed the data on an interactive map with layers for different resource types  
- Integrated the Gemini API to generate real-time updates and review summaries  

The focus was on creating clean, standardized, and maintainable data pipelines that connect open data to a responsive mapping interface.
